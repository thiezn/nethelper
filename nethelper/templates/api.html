NetHelper API Documentation

Someday this is the page that contains extensive documentation about the nethelper API on http://mortimer.trafficmanager.net/api/v1/

For now you'll have to make do with a dump of a few example queries you can do.

Metric Calculations

  • 1024 kbps
  • POST
  • Payload {"kbps": 1024}
  • URL: http://mortimer.trafficmanager.net/api/v1/metrics/calculate
  • 1024 kbps non-rounded
  • POST
  • Payload {"kbps": 1024, "round_to": null}
  • URL: http://mortimer.trafficmanager.net/api/v1/metrics/calculate

Network calculations

  • 10.0.0.0 bitmask 24
  • POST
  • Payload: {"network": "10.0.0.0", "bitmask": 24}
  • URL: http://mortimer.trafficmanager.net/api/v1/networks/calculate
  • 10.0.0.0 netmask 255.255.255.0
  • POST
  • Payload: {"network": "10.0.0.0", "netmask": "255.255.255.0"}
  • URL: http://mortimer.trafficmanager.net/api/v1/networks/calculate
  • 10.0.0.0/24
  • POST
  • Payload: {"network": "10.0.0.0/24"}
  • URL: http://mortimer.trafficmanager.net/api/v1/networks/calculate

MAC Address parsing

  • 00.11.22.33.44.55
  • POST
  • Payload: {"mac_address": "00.11.22.33.44.55"}
  • URL: http://mortimer.trafficmanager.net/api/v1/mac
  • 0011.2233.4455
  • POST
  • Payload: {"mac_address": "0011.2233.4455"}
  • URL: http://mortimer.trafficmanager.net/api/v1/mac
  • 00-11-22-33-44-55
  • POST
  • Payload: {"mac_address": "00-11-22-33-44-55"}
  • URL: http://mortimer.trafficmanager.net/api/v1/mac
  • 00:11:22:33:44:55
  • POST
  • Payload: {"mac_address": "00:11:22:33:44:55"}
  • URL: http://mortimer.trafficmanager.net/api/v1/mac
  • 001122334455
  • POST
  • Payload: {"mac_address": "001122334455"}
  • URL: http://mortimer.trafficmanager.net/api/v1/mac

Ports

  • tcp/22
  • POST
  • Payload: {"port": 22, "protocol": "tcp"}
  • URL: http://mortimer.trafficmanager.net/api/v1/ports
  • udp\21
  • POST
  • Payload: {"port": 21, "protocol": "udp"}
  • URL: http://mortimer.trafficmanager.net/api/v1/ports
